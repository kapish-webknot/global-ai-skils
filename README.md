# global-ai-skills

A collection of Claude-optimized skill packs for various tech stacks.

## CLI Tool

Generate skill packs using the interactive CLI tool:

```bash
cd cli
npm start
```

Or run directly:

```bash
node cli/index.js
```

### Supported Tech Stacks

| Tech Stack | Sub-skills |
|------------|------------|
| React Native | performance-optimization, upgrade-workflows, debugging, navigation, state-management, testing, native-modules, animation |
| Node.js | api-design, authentication, database-integration, testing, security, performance, error-handling, deployment |
| Python | web-development, data-science, testing, async-programming, packaging, performance, security, api-development |
| TypeScript | type-system, generics, decorators, module-system, testing, migration-from-js, advanced-patterns, performance |
| Next.js | app-router, server-components, data-fetching, authentication, seo, performance, deployment, api-routes |

## How to Use Generated Skills with Claude

1. Run the CLI tool to generate a skill pack
2. Open the generated `.md` files
3. Copy the relevant skill content
4. Provide it as context to Claude
5. Ask specific questions related to the skill area
